Title: OpsPulse — Unified Business Health Command Center (High-Fidelity React/Next.js Frontend)
Design Language (The "Unique Factor"):
Theme: "Obsidian Tactical." Background is a deep matte ink (#0a0a0c).
Visual Style: Use Glassmorphism (semi-transparent backgrounds with background-blur). Borders should be thin (1px) with a subtle metallic gradient.
Typography: Use a professional Sans-serif (Inter) for UI and a Monospaced font (JetBrains Mono) for all live data/numbers to give it a "Terminal" feel.
Grid System: 12-column grid with a faint, low-opacity "scanning" grid pattern in the background.
Core Functional Components to Build:
Global "Nervous System" Header:
A top bar featuring a "System Heartbeat" SVG animation (a line that pulses based on real-time data).
A Role Switcher Toggle (Stylized: "Strategic Owner" vs. "Tactical Ops") that smoothly reanimates the dashboard layout using Framer Motion.
A Live Stress Meter (Circular gauge) in the center that glows Red/Amber/Green based on the score.
The "Pulse Dashboard" (Dual Views):
Strategic Owner View: High-level "Impact Cards" showing Runway (Days), Net Margin, and Customer LTV. Use large, bold typography.
Tactical Ops View: High-density "Velocity Charts." Show SKU movement, Support ticket wait times, and Logistics status. Use small, sharp sparklines.
Predictive Stress Engine (The "Time Traveler"):
A main center chart (Area chart) showing current health.
Innovation: Add a "Future Projection" section where the line becomes dotted and extends into a red-shaded area (indicating predicted stockouts or cash drops).
Root Cause "Nerve Feed":
A sidebar showing a real-time stream of business events (e.g., "[12:04] Order #204 Processed").
Visual Logic: When an alert pops up, draw a visible SVG "Nerve Line" (a glowing connection line) that physically links the Alert to the specific data point on a chart that caused it.
Three-Tier Alert System (Cards):
Crisis (Red): Pulsing border, high-priority icons, contains a "Recommended Action" button (e.g., "Reorder SKU").
Opportunity (Green): Glowing emerald shadow, shows ROI potential (e.g., "Scale Ad Spend").
Anomaly (Amber): Subtle flicker effect, indicates deviations.
"War Room Mode" (The Showstopper):
Include a toggleWarRoom function. When active:
Background dims to 95% black.
A red "Scan-line" animation passes top-to-bottom every 5 seconds.
All "Good" news is hidden. Only the "Crisis Cards" and the "Decision Recommendation Engine" (large, centered action buttons) are visible.
Text says: "CRITICAL STATE: ACTION REQUIRED."
Decision Recommendation Engine:
Floating "Action Cards" that pop up near crises.
They must feature a "Simulate Outcome" button that, when hovered, shows how the Stress Score would drop if that action is taken.
Interactivity Requirements:
Use Framer Motion for all transitions (Role switching should feel like a camera panning).
Use Lucide-React for sharp, thin-stroke icons.
Use Tailwind CSS for styling.
No Static Data: All components must accept a "Mock Data Stream" object that updates every 3 seconds to simulate a live backend.
Prompt for the AI Agent: "Build a high-fidelity, high-density React dashboard called OpsPulse using the Obsidian Tactical design language. Use Tailwind and Framer Motion. Ensure the 'War Room Mode' feels like a dramatic UI shift. The UI must look like a premium $500/month SaaS product for elite founders, not a generic open-source template."
